## Introduction

* This datasheet describes the construction of the motor subsystem for the Amphibot V1.

### Project Summary

* Amphibot V1 is modular exploration device that has the ability to go into spaces otherwise inaccessible, it consists of 3pcbs communicating using a 64-byte UART packet structure in a daisy-chain configuration. It streams live FPV video and telemetry to the user while reporting a consolidated hazard score.
